Calculador de Poder — Exportador

Exporte rankings de poder e armazene históricos em CSV.

Arquivos

  • Script: export_power.py
  • requirements.txt
  • Arquivo de saída (power_export.csv) — se presente no repositório

Como usar

Instale dependências e execute o script localmente:

pip install -r requirements.txt
python export_power.py --codes ptb_magnumofspades

Opções úteis:

  • --codes : lista de códigos separados por vírgula.
  • --codes-file : arquivo com um código por linha.
  • --output : arquivo CSV de saída (padrão: power_export.csv).
